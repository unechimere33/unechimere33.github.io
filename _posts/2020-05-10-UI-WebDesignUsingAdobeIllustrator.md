# UI and Web Design Using Adobe Illustrator
<br />
## AI setting for Web Design
- 檔案 > 新增 > 網頁 > 一般 > 建立  
- Illustrator > 偏好設定 > 單位 > { 一般, 筆畫, 文字 > 全選成像素 > 確定 }  
- 視窗 > 工作區 > 網頁

<br />
## Sitemap & Wireframe
- Sitemap 網站地圖：一目瞭然於網站結構、內容。  
- Wireframe 線框圖：使用情境、資訊架構、功能流程等整體性的佈局。  

<br />
## Creating a Sitemap
- File > New > A4 Landscape (Print for client)
- Rectangle Tool, line, Type Tool...

<br />
## Screen Size 版面尺寸
查看各種設備之面板的大小 *<https://www.mydevice.io/>*

<br />
## Create 3 Art boards for user devices (Desktop, tablet, mobile)  
![G01](/assets/UIUsingAI01.jpg)
- 建立第一個面板（電腦）
<br />
新建檔案（Website）> 寬度 { 1200 }  
- 建立第二個面板（平板）
<br />
左邊工具列 [ 工作區域工具 ] > 按上方列 [ 新增工作區域 ] > 寬度 { 768 } 高度任意  
- 第二個面板（手機）
<br />
按Option拖曳複製面板 > 寬度 { 400 }
- 視窗 > 工作區域 > 重新命名3個面板名稱  

<br />
## A responsive 12 column grid
- 畫出一個矩形以覆蓋Desktop工作區域
<br />
矩形工具 { 無填色, 黑線 }  
- 物件 > 路徑 > 分割成網格 > 直欄 { 12 } 間隔 { 30px } 參考線及預視
![G02](/assets/UIUsingAI02.jpg)
- 黑箭頭(V) > 全選 > 解散群組 > 將水平線都刪去 > 下拉增加直欄及工作區域長度
- 全選 > 檢視 > 參考線 > 製作參考線 > 鎖定參考線
![G03](/assets/UIUsingAI03.jpg)
- Art board > 增加工作區域的寬度 (Option Key 對稱加寬)  
![G04](/assets/UIUsingAI04.jpg)

<br />
